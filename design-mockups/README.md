# UWA Website Design Mockups

Three complete design options for the United Wrestling Accord website. Each mockup is a fully functional HTML page with embedded CSS that you can open directly in your browser.

---

## How to View

1. Click on any HTML file below
2. Click "Raw" or "Download" to save it to your computer
3. Open the HTML file in your web browser
4. Or view directly on GitHub by using the Raw link

---

## Design Options

### Option 1: Bold Wrestling Style
**File:** `option-1-bold-wrestling.html`

**Style:** Classic professional wrestling aesthetic with bold, aggressive design elements.

**Key Features:**
- Dark gradient backgrounds (black to dark gray)
- Bold red accent colors and borders
- Large, uppercase typography with heavy letter-spacing
- Wrestling-style scanline effects
- Dramatic text shadows and glows
- 3D-style borders on brand cards with metallic shine effects
- High contrast throughout

**Best For:** Traditional wrestling fans who want that classic "fight night" energy

**Vibe:** WWE Attitude Era / Classic wrestling PPV posters

---

### Option 2: Modern Sports Clean
**File:** `option-2-modern-sports.html`

**Style:** Contemporary sports network design with clean, professional layouts.

**Key Features:**
- Light backgrounds (white and light gray)
- Card-based layouts with subtle shadows
- Modern sans-serif typography
- Gradient headers for each brand card
- Stats section with large numbers
- Clean, organized information hierarchy
- Smooth hover animations

**Best For:** Users who prefer ESPN/sports website aesthetics

**Vibe:** ESPN / Fox Sports / Modern athletic broadcasts

---

### Option 3: Sleek Dark Premium
**File:** `option-3-dark-premium.html`

**Style:** Premium streaming service aesthetic with sophisticated dark mode design.

**Key Features:**
- Deep black backgrounds (#0a0a0a)
- Subtle transparency and blur effects
- Minimalist but luxurious feel
- Accent color system for each brand
- Netflix-style navigation and layout
- Gradient text effects
- Premium typography and spacing
- Understated hover interactions
- Three-column footer with detailed links

**Best For:** Modern streaming audiences expecting Netflix/Disney+ quality presentation

**Vibe:** Netflix / Disney+ / Apple TV+ / Premium streaming platforms

## Design Comparison

| Feature | Option 1: Bold | Option 2: Clean | Option 3: Premium |
